# 💰 Finance Backend API

This is a simple backend project built using **Flask** to manage financial operations.
It is designed as a beginner-friendly backend system and will be expanded with more features.

---

## 🚀 Features

* Basic Flask server setup
* API endpoints for testing
* Structured for future expansion
* Beginner-friendly backend project

---

## 🛠️ Tech Stack

* Python
* Flask

---

```
finance-backend/

├── app.py

```

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/Abhipanda24/finance-backend.git
cd finance-backend
```

2. Install dependencies:

```
pip install flask
```

3. Run the server:

```
python app.py
```
